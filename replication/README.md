Narrow and Wide Replication of Chalfin and McCrary (ReStat 2018)

This code was written as part of a project being prepared to be submitted to the replication section of the Journal of Applied Econometrics. We check for robustness of the results, by replicating the authors' analysis for an extended dataset.


## Replication.R

This file replicate CM's original results as well as outputs the results with the extended dataset. Specifically, the code replicates the summary statistics table, the main results table (Table3), as well as tests of equality of cross crime elasticities (Table6) of CM 2018 for both the narrow and wide datasets. The files 'narrow.csv' and 'wide.csv' consist of the original and extended dataset respectively, while the file 'replicanfuns.R' consists of the funtions needed to implement each of the methods used in the paper.

To implement the replication, follow these steps

1. Go to the folder "replication".
2. Open replication.Rproj
3. Within the project, open replication.R
4. Run the file.
